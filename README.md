# Unitychan-Adventure

* 画面サイズは1920×1080
* HaruPurkon2019.exeを実行することでプレイ可能
* Graphics qualityはHigh以上を推奨（Highより下だとプレイヤーが高速移動してしまう）
* ゲームBGMが大きめなので注意

* zキー：攻撃
* xキー：リロード
